# hypertension-prediction-ml (Still fixing some errors)

🔗 Live App
👉 https://hypertension-ml-prediction.streamlit.app/ 

📌 Overview
This project is a machine learning-powered dashboard that predicts hypertension risk based on lifestyle and health factors.

Users can:
Upload datasets
Explore data visually
Train ML models (Logistic Regression, Random Forest, XGBoost)
Perform personalized risk assessment

⚙️ Tech Stack
Python
Streamlit
Scikit-learn
XGBoost
Pandas, NumPy
Plotly / Matplotlib

📊 Features
Data validation & preprocessing
Exploratory Data Analysis (EDA)
Model training & comparison
Feature importance analysis
Personalized risk prediction

🧠 Machine Learning Models
Logistic Regression
Random Forest
XGBoost
